# Visualizador de Perfil do GitHub

## Descrição

O Visualizador de Perfil do GitHub é uma aplicação web que permite a qualquer usuário pesquisar por um nome de usuário do GitHub e visualizar as informações de seu perfil. A aplicação consome a API do GitHub para obter os dados.

## Funcionalidades

-   **Busca de Usuário:** Permite pesquisar por qualquer usuário do GitHub.
-   **Visualização de Perfil:** Exibe informações como nome, foto de perfil, biografia, número de seguidores e de pessoas que o usuário segue.
-   **Lista de Repositórios:** Mostra os 10 repositórios mais recentes do usuário, com detalhes como nome, estrelas, forks, watchers e a linguagem de programação.
-   **Responsividade:** A aplicação é adaptada para funcionar em diferentes tamanhos de tela.

## Tecnologias Utilizadas

-   **HTML5:** Para a estrutura da página.
-   **CSS3:** Para a estilização e design da aplicação.
-   **JavaScript:** Para a lógica da aplicação e manipulação do DOM.
-   **API do GitHub:** Para a obtenção dos dados dos usuários e seus repositórios.

## Como Usar

1.  Clone este repositório para a sua máquina local.
2.  Abra o arquivo `index.html` em seu navegador de preferência.
3.  Digite o nome de usuário que deseja pesquisar no campo de busca.
4.  Clique no botão "Buscar" ou pressione a tecla "Enter".
5.  As informações do perfil serão carregadas e exibidas na tela.
